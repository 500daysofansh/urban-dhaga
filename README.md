# 🧵 Welcome to URBAN DHAGE

**Urban Dhage** is a modern eCommerce platform focused on stylish and affordable fashion.

---

## 🌐 Live Website  
Check out the live project here:  
👉 https://www.urbandhage.in/

---

## 🚀 About the Project  
Urban Dhage is built to provide a smooth and responsive shopping experience with a focus on:

- 👕 Trendy clothing collections  
- ⚡ Fast and intuitive UI  
- 📱 Fully responsive design  
- 🛒 Seamless browsing experience  

---

## 🛠 Tech Stack  

- Frontend: React / Next.js  
- Backend: Node.js / Express  
- Database: MongoDB / Firebase  
- Styling: Tailwind CSS  

---

## ✨ Features  

- Product listing & categories  
- Dynamic product pages  
- Add to cart functionality  
- Responsive mobile-first design  
- Admin-ready structure (extendable)
